🎓 Student Grade Tracker
A simple Java console application to manage student records and automatically calculate grades using MySQL and JDBC.



✨ What it does
Add and manage student details
Automatically calculates total, average, grade, and pass/fail status
View all students in one place
Find topper and lowest scorer
Generate a quick class performance summary

🛠️ Tech Used
Java, MySQL, JDBC, Maven


⚙️ Setup
Clone the project from GitHub
Create the MySQL database and connect it in your config file
Build and run using Maven or your IDE


👨‍💻 About
A clean beginner-friendly project built to understand Java OOP, database connectivity, and real-world application structure.
